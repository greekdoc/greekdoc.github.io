τε, τέ
  • Parse: particle/Conjunction
  • Meaning:
    • and
    • ἐάν τε = if
    • τε … καὶ … = both … and …
    • τε … δὲ … = both ... and ..
    • τε … δὲ … = on the one hand …, but on the other ..
    • ἐμέ τε ἀποκατασταθῆναι ..., ἐκεῖνον δὲ κρεμασθῆναι
      for me to be restored ... but for him to be hanged (Gen 41:13)
    • ἐάν τε γὰρ … μήτε … = for if … neither …
